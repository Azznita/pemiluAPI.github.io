Apa itu API Pemilu?

Sebanyak 180 juta orang terdaftar sebagai pemilih dalam Pemilu 2014 di Indonesia dan kurang lebih 67 juta dari mereka akan memilih untuk pertama kalinya.

Pemilih butuh jawaban. Apakah saya terdaftar? Bagaimana pendirian para kandidat mengenai permasalahan yang penting bagi saya? Bagaimana mencoblos yang baik dan benar? Bagaimana apabila saya menemui kesulitan untuk memilih?

Untuk menjawab pertanyaan-pertanyaan tersebut, dan masih banyak lagi, kami membantu membangun sebuah tempat penyimpanan informasi terbuka untuk para pemilih Indonesia pada tahun 2014 — sebuah pemilu API.

Developers are using API Pemilu to build mobile and web applications; find the apps at pemiluapps.org and try them on your phone.

Apa itu API?

API adalah singkatan dari Application Programming Interface. Interface atau antarmuka adalah dimana dua sistem bertemu dan berinteraksi. Sebuah API adalah antarmuka yang digunakan oleh sebuah aplikasi agar dapat berkomunikasi dengan aplikasi lain.

API Pemilu 2014 akan menjadi sebuah aplikasi yang menggunakan antarmukanya agar informasi dan fungsionalitas tentang pemilu 2014 di Indonesia tersedia untuk aplikasi lain.
