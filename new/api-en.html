What is API Pemilu?

As many as 187 million people are registered to vote in Indonesia's 2014 elections. An estimated 22 million of those people will be voting for the first time.

Voters need answers. Am I registered? Where do candidates stand on the issues that are important to me? How do I mark the ballot? What can I do if I'm having trouble voting?

To answer those questions, and many more, we've built an open repository of information for Indonesian voters in 2014 — an elections API.

What's an API?

An API is an Application Programming Interface. An interface is where two systems meet and interact. An API is an interface that an application uses to enable other applications to communicate with it.

The Elections API 2014 is an application that uses its interface to make information and functionality about Indonesia's 2014 elections available to other applications.
