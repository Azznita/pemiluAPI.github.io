Kamu bisa menemukan permulaan dari proposal teknis untuk API pada README situs ini di GitHub.

Jika kamu ingin mengobrol lebih lanjut mengenai proyek ini, kamu bisa memulai atau bergabung dalam percakapan di laman GitHub's Issues situs ini.

Atau Anda dapat mengisi survei singkat berikut ini dan berikan saran Anda mengenai bagaimana cara menjangkau lebih banyak orang dengan informasi pemilu yang paling mereka butuhkan.

Di bawah ini adalah beberapa ide aplikasi yang bisa dibuat pada PEMILU API 2014, sebuah API untuk Pemilu 2014 di Indonesia.
